Here are the step-by-step instructions to add your Index.html file to your GitHub repository: 1. **Open your terminal** (Command Prompt, PowerShell, or Git Bash). 2. **Navigate to your repository folder** using the `cd` command. For example: ``` cd path/to/your/repo ``` 3. **Copy or move your file** (`/d:/uiuxlab/Lab1/Index.html`) into the repository folder. You can use File Explorer or a command like: ``` copy /d:/uiuxlab/Lab1/Index.html . ``` (The `.` means current directory.) 4. **Add the file to git staging area:** ``` git add Index.html ``` 5. **Commit the change:** ``` git commit -m "Add Index.html" ``` 6. **Push the commit to GitHub:** ``` git push ``` Your file will now be added to your GitHub repository.
